# Release Notes

## Current Public Source Package

This release contains the public `MMT Ichi Workflow` Pine v6 source package for TradingView.

Included scripts:

- `MMT Ichi Workflow Overlay`
- `MMT Ichi Workflow Strategy`
- `MMT Ichi Workflow Overlay Profiles`
- `MMT IchiOP Crypto Doubled Strategy`
- `MMT Ichi 2D Overlay Profiles`
- `MMT Ichi 2D Overlay Profiles Strategy`

## Highlights

- Adds profile-aware strategy validation for the profile overlay model.
- Adds a separate 2D overlay and strategy pair for intentional 2D chart review.
- Includes public optimizer source files and example configuration for reproducible Ichimoku settings research.
- Keeps generated optimizer run outputs out of source control.

## Notes

- These scripts are intended for educational and research use.
- Strategy results depend on ticker, timeframe, TradingView data availability, settings, and Strategy Tester assumptions.
- Always review the matching strategy layer before using an overlay for chart-side execution decisions.
